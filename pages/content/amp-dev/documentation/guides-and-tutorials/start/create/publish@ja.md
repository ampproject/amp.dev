ローカルでページをテストをして、エラーの修正が完了したら、初めての AMP ページを配信する準備は完了です。

AMP の動作や各要素の振る舞いについて詳しく学ぶには、左にある見出しから詳細ページをご確認ください。コンテンツの制作に役立つツールの基本情報を知りたい方は、[開発ワークフローを開始する](https://developers.google.com/web/tools/setup/)をご覧ください。
