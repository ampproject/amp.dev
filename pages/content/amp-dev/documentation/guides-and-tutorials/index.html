Introduction to AMP Websites

The AMP project is a web components framework to create lightning fast, user first, immersive websites. By building on web development best practices, AMP makes it easy to build beautiful and functional sites that just work. AMP is cross browser compatible and supported by many different platforms.

{% do doc.styles.addCssFile('/css/components/atoms/format-tags.css') %}

Today, AMP can build responsive and interactive websites, immersive full-screen stories, secure and engaging ads and dynamic email.
