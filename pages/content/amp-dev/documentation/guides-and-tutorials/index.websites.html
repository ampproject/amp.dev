Get started with AMP

AMP is a web components framework to create lightning fast, user first, immersive websites. By building on web development best practices, AMP makes it easy to build beautiful and functional sites. AMP is cross browser compatible and supported by many different platforms.

To get started, follow our tutorial on how to Create your first AMP Page and learn How to Convert HTML to AMP, then familiarize yourself with the rest of the guides & tutorials, organized into stages of development:
